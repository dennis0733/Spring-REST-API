# Spring-REST-API
A Product Spring Boot Rest-Api Service

Running Instructions:

1. You need to create a new database named 'productservice' through MySQL DB.


![AdsÄ±z](https://github.com/dennis0733/Spring-REST-API/assets/69004609/c2e62f6f-eedb-474d-b6bc-edea925f5aff)
